# Free Fire Tournament API (Scaffold)

Generated by ChatGPT. Minimal Express + Mongoose scaffold including models, basic controllers, and routes.

## How to run
1. Extract the zip.
2. Copy `.env.example` to `.env` and set `MONGO_URI` and `JWT_SECRET`.
3. Install dependencies:
   ```
   npm install
   ```
4. Start:
   ```
   npm run dev
   ```
5. API root: `http://localhost:3000/`

## What's included
- Mongoose models for AdminUser, Player, Team, Tournament, Registration, Match, Bracket, Transaction, Coupon, Report, AntiCheatLog, AuditLog, Notification
- Basic admin auth middleware (JWT)
- Admin routes + controller stubs
- Bracket service placeholder
- Payment & notification placeholders

You can expand controllers and add payment/webhook integrations.
